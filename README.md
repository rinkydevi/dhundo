# dhundo

**dhundo** (Hindi: ढूंढो — *to find*) helps you discover the right AI tool for any task, instantly.

Describe what you want to do — edit reels, write a blog, remove a background — and dhundo matches you to the best AI tool for it, with India-specific details like UPI support, pricing in INR, and VPN requirements.

## Live

[https://dhundo.vercel.app](https://dhundo.vercel.app)

## Features

- Natural language search — describe your task, not the tool
- India-first: UPI support, INR pricing, VPN flags, accessibility info
- Free tier indicators
- Top 3 ranked matches with reasons why each fits your query

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS v4
- Deployed on Vercel
